# Projeto de Atividades - Seguranca na Informacao

Este repositorio reune as atividades desenvolvidas para a disciplina de Seguranca na Informacao. O material esta organizado por pasta, com uma entrega principal para cada atividade.

## Estrutura do repositorio

### `Atv01`
Contem a atividade de modelagem de ameacas do TCC `GEOSEGMENT`, com uso das metodologias STRIDE e DREAD para identificar, classificar e priorizar ameacas.

Arquivo principal:
- `Atv01/Levantamento_de_ameaças_Segurança_na_Informação_GEOSEGMENT (1).pdf`

### `Atv02`
Contem uma atividade pratica de verificacao de integridade por hashes, com script em Python, relatorio atual e historico de execucoes.

Arquivo principal:
- `Atv02/README.md`

### `Atv03`
Contem a analise do TCC sob a otica do OWASP Top 10, indicando quais categorias podem ser aplicadas ao sistema e justificando cada escolha.

Arquivo principal:
- `Atv03/Análise_TCC_sobre_a_ótica_OWASP_TOP_10.pdf`

## Objetivo do projeto

O objetivo deste repositorio e centralizar as entregas academicas da disciplina, facilitando a visualizacao do conteudo produzido e a organizacao do portfolio.

## Observacao

A atividade sobre OWASP Top 10 esta armazenada na pasta `Atv03`, enquanto a pasta `Atv02` ja possui outra entrega documentada em seu proprio `README.md`.
