# Atv01 - Modelagem de Ameacas do TCC

Esta atividade apresenta a modelagem de ameacas do projeto `GEOSEGMENT`, utilizando duas abordagens classicas de seguranca:

- `STRIDE`, para classificar os tipos de ameaca.
- `DREAD`, para avaliar e priorizar os riscos identificados.

## Objetivo

Identificar possiveis ameacas ao sistema do TCC, entender seu impacto no contexto do projeto e organizar a analise de forma estruturada.

## O que foi desenvolvido

O documento da atividade contempla:

- levantamento das ameacas relacionadas ao sistema;
- classificacao das ameacas com base no modelo STRIDE;
- avaliacao da gravidade e priorizacao usando DREAD;
- organizacao da analise em formato de entrega academica.

## Arquivo da atividade

- `Levantamento_de_ameaças_Segurança_na_Informação_GEOSEGMENT (1).pdf`

## Resumo da proposta

A atividade busca demonstrar como tecnicas de modelagem de ameacas podem ser aplicadas a um projeto real, ajudando a antecipar riscos e apoiar decisoes de seguranca ainda nas fases de analise e projeto.
